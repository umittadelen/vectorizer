Image
Brightness 1.00
Overall lightness. Below 1 darkens the image; above 1 brightens it. Applied first before any other step.
Gamma 1.00
Mid-tone curve. Below 1 lifts faint grey lines (reveals hidden detail); above 1 pushes midtones darker (kills grey noise).
Contrast 1.00
Boosts separation between ink and paper before threshold. Helps junction areas stay defined without blending into each other.
Sharpen 0
Unsharp mask applied before threshold. Reinforces thin lines and cyclic junctions so potrace traces them as distinct shapes rather than blending them.
Bitmap
Dilate 0
Expand bright (white) areas N pixels. Thickens lines and closes small gaps between strokes before threshold.
Erode 0
Shrink bright (white) areas N pixels. Thins lines and removes isolated 3×3 dot clusters before tracing.
Threshold 100
Pixels brighter than this become ink. Lower = more ink captured; higher = only the brightest lines.
Pre-blur 0.00
Gaussian blur before threshold. Softens jagged edges so lines trace smoother. 0 = off.
Break Corner Joins
Removes pixels where two regions only touch diagonally. Separates hourglass / bowtie junctions into distinct traced paths.
Potrace
Alphamax 1.30
Curve smoothness (0–1.333). 0 = all sharp corners; 1.333 = maximum rounding.
Opt Tolerance 0.30
Curve simplification. Higher = fewer nodes, smoother. Lower = follows lines more tightly.
Turd Size 0
Remove blobs smaller than N pixels. Kills noise/speckles. 0 = keep everything.
Output
Invert Colors
On = white lines on black. Off = black lines on white (easier to print).
